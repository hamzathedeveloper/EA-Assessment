Steps to Run the Project:

1. Clone the project.
2. Set your connection string in appsettings.json of Project 'EmiratesAuctionDataAPI'
3. Set 'MyApiUrl' with local URL of EmiratesAuctionDataApi in appsettings.json of Project 'EmiratesAuction'.
3. Run command 'Add-Migration xyz' on EmiratesAuctionDataApi
4. Run command 'update-database' on EmiratesAuctionDataApi
5. Run both the projects and the data in the tables will be seeded on initialization.
